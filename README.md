# [roadmap](http://roadmap.rectenna.net)

+ indoor, 1 year deployment
+ outdoor, 2 years deployment
+ drones, 5 years deployment
+ space, 10 years deployment


## indoor

### specification

+ distance: 10m
+ band: wifi 6e, wifi 7
+ frequency: 2.4 Ghz, 5Ghz, 6-7Ghz

### area:

+ Industry

### use case:

+ supertag
+ eInk screens
+ wireless price tag with a screen
+ batteryies AA / AAA replacement in low power devices



## outdoor


### specification:

+ distance: 100m
+ band: 5G
+ frequency: 26-35Ghz


### area:

+ Transportation:
  + refrigerated vehicles
+ Smart City:
  + sensors
+ Wearable  


### use case:

+ supertag
+ IoT sensors






## drones

### area:

+ drones
+ mobile vehicles




## satellites

### specification:
